# Lost & Found App

A full-stack **Lost & Found** web application built with **MERN (MongoDB, Express, React, Node.js)**. Users can register, log in, post lost or found items, and mark items as returned. The app supports filtering, searching, and real-time updates.

## Demo

- **Frontend:** [https://full-mern-stack-project-3.onrender.com](https://full-mern-stack-project-3.onrender.com)  
- **Backend:** [https://full-mern-stack-project-1.onrender.com](https://full-mern-stack-project-1.onrender.com)  

---

## Features

- User authentication with JWT
- Add lost or found items
- Mark items as returned
- Filter items by type (Lost/Found) and status (Open/Returned)
- Search items by title
- View items posted by yourself
- Responsive UI built with React and TailwindCSS
- RESTful API

---

## Tech Stack

- **Frontend:** React, TailwindCSS, Axios, Vite
- **Backend:** Node.js, Express, MongoDB, Mongoose, JWT, CORS
- **Deployment:** Render (Frontend & Backend)
